# ❤️ Heart Attack Prediction System

An AI-powered web application that predicts the likelihood of a heart attack using machine learning algorithms and patient health data.

> ⚠️ This project is intended for educational and research purposes only. It is **not** a substitute for professional medical diagnosis.

---

## ✨ Features

- 🩺 Predicts heart attack risk in seconds
- 🤖 Machine Learning based prediction
- 📊 User-friendly input form
- ⚡ Instant results
- 💻 Responsive web interface
- 🔒 Runs locally without internet

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Afmtechz/Heart-Attack-Prediction-System.git
```

Move into the project

```bash
cd Heart-Attack-Prediction-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
